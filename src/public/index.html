📚 Documentação da API

Bem-vindo à API de Fornecedores. Abaixo estão listados os endpoints disponíveis para consumo.

📦 Fornecedores

  • GET /api/v1/fornecedores - Listar todos os fornecedores
  • GET /api/v1/fornecedores/:id - Buscar fornecedor por ID
  • POST /api/v1/fornecedores - Criar novo fornecedor
  • PATCH /api/v1/fornecedores/:id - Atualizar fornecedor
  • DELETE /api/v1/fornecedores/:id - Remover fornecedor

🔐 Autenticação (Auth)

  • POST /api/v1/users/signup - Criar conta (Cadastro)
  • POST /api/v1/users/login - Entrar no sistema (Gera Token JWT)
  • POST /api/v1/users/esqueciSenha - Solicitar link de recuperação de senha
  • PATCH /api/v1/users/resetarSenha/:token - Redefinir senha usando o token recebido
  • PATCH /api/v1/users/atualizarSenha - Logado: Alterar senha atual 🔒

👤 Usuários (Perfil)

  • GET /api/v1/users/ - Retorna todos os usuarios cadastrados
  • PATCH /api/v1/users/atualizarUsuario - Atualizar dados cadastrais (Nome/Email) 🔒
  • DELETE /api/v1/users/deletarUsuario - Desativar conta (Exclusão Lógica) 🔒

🔒 = Requer Token Bearer no Header Authorization.
